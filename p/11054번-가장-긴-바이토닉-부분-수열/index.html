11054번 가장 긴 바이토닉 부분 수열

다이나믹

Dec 21, 2020

일단 C++ 이 내 주 언어니까 vsc 에 C/C++ 환경을 따로 마련을 해야 할텐데.. 이쪽 관련 내옹은 dev 카테고리에 따로 적는걸로 하겠다. 분량이 생각보다 많아질 것 같다.

오늘은 시간이 그리 많지 않으니, 라즈비안을 깔 때 기본적으로 제공해주는 Geany 라는 ide 를 이용해보겠다. 기가 지니 아니다. 아직 자세하게는 모르지만, 이걸로도 충분할 것 같다.

문제

https://www.acmicpc.net/problem/11054

문제를 요약하자면 다음과 같다. 수열이 주어지면 오름차순으로 한 번, 내림차순으로 한 번 훑었을 때 최장 수열 길이를 구하시오. (이게 이해가 안가면 링크를 타고 들어가 읽어보는 걸 추천한다.)

10
1 5 2 1 4 3 4 5 2 1

7

여기서의 바이토닉 부분 수열은 1 2 3 4 5 2 1 이다.

풀이과정

보고 바로 생각난 풀이는 수열을 전부 돌아보고 최댓값들을 찾은 뒤, 그 최대값의 양 옆으로 LCS(최장 증가 수열)를 구한 뒤 그 두개의 길이를 더해서 최댓값을 정하는거다.

그렇게 되면 이제 이런 케이스가 날 반긴다.

10
1 2 3 4 5 4 3 2 1 10

9

내가 생각한 풀이대로 풀면 이게 6이 나올 것이다. 1 2 3 4 5 10 이런식으로. 그래서 이걸 어떻게 풀어야 하지.. 하고 생각하던 찰나 굇수 JK 님이 풀이를 갑자기 딱 주셨다. (위의 케이스도 JK 님이 주신 것이다. 무한한 감사를 표합니다)

dp 배열 두개를 만들어서 하나는 오름차순, 다른 하나는 내림차순으로 만든 뒤 각 인덱스의 값을 더해주면 된다는 것이다. (여기에 대해 부가설명을 해야 하나 말아야 하나 고민을 좀 해봤는데, 그렇게 계속하다 보면 밑도 끝도 없을 것 같아서 생략하겠다. 알아서 이해하자.)

맞다. 드럽게 쉬운 문제다. 이런 문제도 파악 못하는 자신을 반성해야 한다.

소스 코드는 다음과 같다.

#include <stdio.h>
int n,m;
int a[1001];
int d1[1001],d2[1001];
int main(){
    scanf("%d",&n);
    for (int i=0;i<n;i++)
        scanf("%d",&a[i]);
    
    for (int i=0;i<n;i++)
        for (int j=0;j<i;j++)
            if (a[i]>a[j] && d1[i]<d1[j]+1)
                d1[i]=d1[j]+1;
    
    for (int i=n-1;i>=0;i--)
        for (int j=n-1;j>i;j--)
            if (a[i]>a[j] && d2[i]<d2[j]+1)
                d2[i]=d2[j]+1;
    
    for (int i=0;i<n;i++){
        if (m<d1[i]+d2[i]+1)
            m=d1[i]+d2[i]+1;
    }
    printf("%d",m);
    
    return 0;
}

혹 이해가 되지 않는다면 언제든지 질문해줬으면 좋겠다. 그게 나에게도 공부가 될성 싶다.

되도록 문제는 꼭 내 힘으로 풀도록 하자. 다른 사람 도움 없이. 그래야 실력이 는다.
